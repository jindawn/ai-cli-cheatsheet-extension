# AI CLI 速查表隐私政策

生效日期：2026 年 7 月 14 日

AI CLI 速查表的 Chrome Web Store 版本仅在用户的浏览器中运行，不向项目维护者或任何第三方服务器传输数据。

## 本地处理的数据

扩展会在浏览器本地保存以下内容，用于提供速查和偏好设置：

- 搜索词和所选筛选条件。
- 收藏的条目标识。
- 最近复制的条目和命令，最多 20 条。
- 平台、已启用工具和引导状态等偏好。

这些数据仅保存在 `chrome.storage.local` 中，不会离开用户的设备。扩展不读取浏览历史、网页内容、Cookie、账号信息或身份信息。

## 数据共享与用途

项目不收集、出售、出租、分享或用于广告、信用评估及其他与速查功能无关的用途。

## 保留与删除

本地数据会保留到用户在扩展中清空对应记录，或卸载扩展并删除其浏览器存储数据。

## 源码维护版说明

GitHub 源码版可由高级用户单独安装 Native Host，用自己的 Claude Code 或 API 配置维护本地数据。该功能不包含在 Chrome Web Store 版中，其数据处理和费用取决于用户自己选择的服务。

## 联系方式

如对本政策有疑问，请通过 [GitHub Issues](https://github.com/jindawn/ai-cli-cheatsheet-extension/issues) 联系项目维护者。
